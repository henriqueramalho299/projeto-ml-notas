# 📊 Previsão de Desempenho de Estudantes com Machine Learning

Projeto desenvolvido em Python com o objetivo de prever a nota de estudantes com base em diferentes fatores.

## 🧠 Variáveis utilizadas
- Horas de estudo
- Horas de sono
- Número de faltas

## 🚀 Tecnologias
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 📈 O que foi aplicado
- Regressão linear
- Divisão treino/teste
- Avaliação de modelo
- Visualização de dados
